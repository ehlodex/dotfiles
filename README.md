# dotfiles

## Installation

Clone this repo into `~/.dotfiles`:

    git clone https://github.com/ehlodex/dotfiles.git ~/.dotfiles

Link the dotfiles:

    cd ~/.dotfiles
    bash dots

## vim 8+

Install gruvbox:

    mkdir --parent ~/.vim/pack/morhetz/start
    git clone https://github.com/morhetz/gruvbox.git ~/.vim/pack/morhetz/start/gruvbox
